## General UI/UX of legal software products :  

We went through 10 of other legal web portals and studied their UI/UX. This document contains all the insight we got out of those.

1 - [Osler](https://www.osler.com/en/) :<br/>
    Theme: red, white, black <br/>
    The website uses a hero section, with red parts in other sections, and a black footer.
	Font: Noto Sans.<br/>
	Animation: carousel, smooth loading, use gifs for carousel.<br/>

2 - [goldblattpartners](https://goldblattpartners.com/):<br/>
    Theme: red, white, gray
    The website uses gray boxes for each section, a white and red footer<br/>
    Font: Roboto<br/>
    Animation: none<br/>

3 - [steinbergsllp](https://steinbergsllp.com/):<br/>
    Theme: blue, some red (not a lot), white, dark blue
    The website uses a hero section with carousel, a dark blue footer<br/>
    Font: Lato<br/>
    Animation: on scroll showing a box of data and also calculating dollars<br/>

4 - [mcmillan](https://mcmillan.ca/):<br/>
    Theme: red, white, some boxes dark gray
    The website uses most sections light gray or white, hero section a dark blue image and carousel, footer white and red<br/>
    Font: Montserrat<br/>
    Animation: lazy loading on scroll, carousel<br/>

5 - [emecorp](https://www.emecorp.ca/):<br/>
    Theme: gold, dark color, black, white
    The website uses a black footer, white header, and images with golden borders<br/>
    Font: Muli<br/>
    Animation: none<br/>

6 - [clio](https://www.clio.com/):<br/>
    Theme : dark blue, white, gray<br/>
    The website uses white background with dark-blue header and gray footer.<br/>
    Font : "Poppins" for headers and "Source Sans Pro" for paragraph text.<br/>
    Animation : None<br/>

7 - [spellbook](https://www.spellbook.legal/):<br/>
    Theme : Black, red, blue, yello, purple
    This website has a black background and the text is written in gray font color. Each section has its own color coding.<br/>
    Font : Arizona Mix<br/>
    Animation : Lottie animations on headers of each section.<br/>

8 - [harvey](https://www.harvey.ai/):<br/>
    Theme : This website is mostly in black and white.<br/>
    Font : HarveySerifFont (it's a custom font)<br/>
    Animation : zoom in on pictures<br/>

9 - [thomsonreuters](https://www.thomsonreuters.ca/):<br/>
    Theme : Sections' backgrounds are either white or gray.
    Important buttons are red<br/>
    Font : Clario<br/>
    Animation : Nothing<br/>

10 - [avocato](https://avocato.ca/):<br/>
    Theme : black - purple -pink<br/>
    Font : Darker Grotesque<br/>
    Animation : Vertical slide in/out for sections of new information.<br/>
